# 从零搭建react-ssr应用

我觉得这样的一个项目能够很好将我目前所学到的知识糅合起来

- webpack的配置
- node的服务端应用
- react在服务端中如何编写业务代码
- 路由的配置和redux是如何通信的

在中间层编写代码需要注意什么到页面展示我们需要准备大量的工作，所以本项目只是一个纯粹的学习和采坑的过程
